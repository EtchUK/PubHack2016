Drop svg icons and images in here. They get compiled into a svg sprite that's included inline in the page.
Useful articles
https://css-tricks.com/svg-sprites-use-better-icon-fonts/
http://tympanus.net/codrops/2015/07/16/styling-svg-use-content-css/

To use an icon, once the svg file has been put in the /svgs folder:
<icon name="'file-name-no-extension'"></icon>
